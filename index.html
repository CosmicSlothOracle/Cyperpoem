Methodology & Discovery

Classification: Authorized Lab Assessment

Initial Findings

  • Passive Recon: Target identified via subfinder (careerportal, ai-challenge, chat). DNS TTL of 300 implies DDoS hardening.
  • TCP Sweep: Primary port scan via naabu (SYN scan, moderate rate) verified 5 open ports: 21, 22, 80, 1080, 3389.
  • UDP Sweep: Mostly open|filtered due to no-response. Port 520 definitively closed (ICMP port-unreachable).

Following the discovery phase, service-aware verification was executed on confirmed open TCP ports using custom async Python probes and deep nmap fingerprinting.
